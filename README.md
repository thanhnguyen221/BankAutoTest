## ✨ Key Features
- **Visual AI Tracking**: Features a "Flying Head" animation that physically moves to and interacts with clicked elements.
- **Dynamic Data Entry**: Realistic "typewriter" effect for inputting data into forms.
- **Smart Tab Handling**: Logic-aware navigation that prevents misclicks between "Bank" and "Branch" tabs.
- **Marker Validation**: Real-time verification of Leaflet map markers after successful CRUD operations.

## 🛠 Tech Stack
- **Language:** Python
- **Automation:** Selenium WebDriver
- **Web Framework:** Django
- **Maps:** Leaflet.js
